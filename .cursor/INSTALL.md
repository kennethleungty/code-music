# Installing claude-music for Cursor

## Steps

1. Clone this repository into your Cursor extensions or project directory:

```bash
git clone https://github.com/kennethleungty/claude-music.git
```

2. In Cursor's Agent chat, run:

```
/add-plugin claude-music
```

Or search for **claude-music** in the Cursor plugin marketplace.

3. Start a new session and try:

```
/play
```

## Requirements

- An audio player: **mpv** (recommended) or **ffplay**
- The plugin auto-detects your platform and walks you through installing one if needed
