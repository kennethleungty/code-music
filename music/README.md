# Fallback Music Files

Place royalty-free MP3 files here as offline fallbacks. These are used when internet radio streams are unavailable.

## Naming Convention

```
lofi-fallback.mp3
jazz-fallback.mp3
classical-fallback.mp3
ambient-fallback.mp3
```

The controller matches files by genre prefix, so `lofi-chill.mp3` or `lofi-beats.mp3` also work.

## Requirements

- Royalty-free / Creative Commons Zero (CC0) licensed
- MP3 format
- Keep files small (under 500KB each, 30-60 second loops ideal)

## Sources for Free Music

- [freepd.com](https://freepd.com/) — CC0 public domain music
- [pixabay.com/music](https://pixabay.com/music/) — royalty-free music
- [incompetech.com](https://incompetech.com/) — CC BY music by Kevin MacLeod
